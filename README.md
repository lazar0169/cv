# Portfolio Website

This is a personal portfolio website built using HTML, CSS, and JavaScript. It showcases my projects, skills, and contact information.

## Features

- **Responsive Design**: Looks great on both desktop and mobile devices.
- **Dynamic Content**: Uses a JSON configuration file for easy content updates.
- **Smooth Animations**: Enhances user experience with subtle animations.

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Edit `portfolio-config.json` to customize your content
